# Using MLB Statcast Jump Metrics to Evaluate Outfield Defense

This repository details a project I did during my junior and senior years at Rice University, titled *Using Statcast Jump Metrics to Evaluate Outfield Defense*. I started working on this as a final project for SMGT 435: Baseball Analytics in Fall of 2024, which was my junior year.  After presenting it in that class, I worked with the Professor for that class, Dr. Scott Powers. to expand on my findings and eventually present them at Saberseminar 2025 in Chicago, Illinois. 

The project fits a logistic mixed-effects regression model on several MLB statcast "jump" metrics for defensive outfielders including Jump, Burst, Route, and Sprint Speed, and tries to predict catch probability. Using the results of this model, we can compare the relative ranges of different players in the outfield and evaluate who is over or underperforming. The model was built using 2024 regular season data, so keep that in mind when considering results about individual outfielders' performances in the most recent year. The key finding from the project was that Burst, which is a proxy for acceleration, is significantly more important in making a catch in the outfield than the outfielder's route efficiency or raw speed.
